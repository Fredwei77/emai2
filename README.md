# 现代化网站项目

这是一个使用现代前端技术栈构建的响应式网站项目。

## 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代样式
- **JavaScript (ES6+)** - 交互逻辑
- **Tailwind CSS** - 实用优先的CSS框架
- **Vite** - 快速构建工具

## 项目结构

```
website/
├── src/                    # 源代码目录
│   ├── index.html         # 主页
│   ├── css/               # 样式文件
│   │   ├── main.css       # 主样式文件
│   │   └── components.css # 组件样式
│   ├── js/                # JavaScript文件
│   │   ├── main.js        # 主脚本文件
│   │   └── components.js  # 组件逻辑
│   └── assets/            # 静态资源
│       ├── images/        # 图片文件
│       └── icons/         # 图标文件
├── public/                # 公共文件
├── dist/                  # 构建输出目录
└── package.json           # 项目配置
```

## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 开发环境要求

- Node.js 18+
- npm 或 yarn

## 功能特性

- 响应式设计
- 现代化UI
- 快速构建
- 代码分割
